Project1- EVL - Sweet Home Chicago

Project one under Visualization and Visual Analytics class we had to create a map of chicago and describe the demographics of the CHICAGO using the D3JS library with Javascript as the main language of use. The project deals with trying to uncover the basic trends and patterns that emerge through the demographics of Chicago based on different parameters.
The project deals with the following parameters for identifying demographics :
  • Age and Gender in Chicago
  • Race by Population
  • Race and Age
The D3 library was created by Mike Bostock and we are using the extension of that. Mike's git has been immensely helpful to learn and understand the concepts of D3JS and which has helped me get my application up and running. Alongside, the following websites have been helpful.
